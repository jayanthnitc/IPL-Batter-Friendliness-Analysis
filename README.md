# IPL Batting Era Analysis

## Project Overview

This project investigates whether the IPL has become increasingly batter-friendly over time.

The analysis focuses on:

- Average scores
- Run rates
- Sixes
- Bowling economy
- 200+ totals
- Venue impact

## Objective

To determine whether statistical evidence supports the claim that modern IPL cricket favors batters more than earlier seasons.
